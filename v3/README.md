# Nowhere Craft Website

## Overview
A premium, dark-themed website designed for "Nowhere Craft". The design focuses on "Underground Luxury" using a palette of deep charcoal, warm off-white, and copper accents.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Grid, Animations.
- **JavaScript**: Vanilla JS for interaction (scroll reveals, custom cursor).
- **Fonts**: 'Grenze Gotisch' (Modern Gothic) and 'Inter' (Clean Sans-serif) via Google Fonts.

## How to View
1. Open `index.html` in your web browser.
2. Navigate through the menu to explore other pages (`About`, `Collections`, `Journal`, `Contact`).

## Design Features
- **Visuals**: Dark mode default with a subtle noise texture overlay for a tactile feel.
- **Typography**: Detailed typographic hierarchy with a custom blackletter-inspired heading font.
- **Interactions**:
    - Custom cursor with magnetic hover effects.
    - Reveal-on-scroll animations for content.
    - Smooth text transitions.
- **Responsiveness**: Fully responsive grid layout that adapts to mobile devices.

## File Structure
- `index.html`: Home page.
- `about.html`: Philosophy and brand story.
- `products.html`: Product collection grid.
- `journal.html`: Blog/Stories list.
- `contact.html`: Contact form.
- `css/style.css`: Global styles and animations.
- `js/script.js`: Global interaction logic.
- `assets/images/NOWHERE_yatay_1000px.png`: Brand logo.

## Customization
- **Colors**: Edit the `:root` variables in `style.css` to adjust the color palette.
- **Content**: Update the HTML files with real product images and copy.
